View Source and note ng-app on html tag of page. This tells Angular to control this entire page

ng-controller

View Source and note ng-controller on body tag of page. This tells Angular to use a specific controller for elements within the body. The actual controller is loaded via javascript in the head section. For more complete pages, the controller would be applied to more specific elements of the page.
